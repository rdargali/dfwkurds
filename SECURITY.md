# Security

If you discover a security vulnerability, please do not open a public issue. Instead, contact the maintainers privately so the issue can be handled responsibly.

Recommended steps

- Email: security@your-domain.example (replace with an actual contact)
- Provide a clear summary, steps to reproduce, and potential impact.
- Avoid sharing exploit code publicly until a fix is released.

After reporting

Maintainers will acknowledge receipt and provide an estimated timeline for a fix. Once a fix is available, the issue will be disclosed publicly with credit to the reporter, unless they request anonymity.
