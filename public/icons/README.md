# PWA Icons

This directory should contain PWA icons in the following sizes:

- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`

## Generating Icons

See `/scripts/generate-icons.md` for instructions on generating these icons.

## Placeholder

Until proper icons are created, you can use a simple colored square or the Kurdish Sun symbol as a placeholder. The icons will be generated automatically during the build process if you have a source image.
